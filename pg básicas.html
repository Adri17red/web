1.Contenido principal

El contenido principal de la página se agrupa en una única caja que no se repite en otras páginas. Debe contener lo más importante y no puede estar anidado dentro de otra estructura. Estructura: "main"
    Características:
  • Sólo puede haber un único main por página
  • No puede estar dentro de otra página
  • Dentro contendrá aquel contenido que no se repita en cada página

2.Encabezado

El encabezado contiene el título o presentación de la página. Suele incluir logotipos, menús o elementos introductorios. Estructura: "header"

3.Pie de página

Aquí se coloca la información final como autoría, fecha o enlaces legales. Es común en todas las páginas del sitio. Estructura: "footer"

4.Enlaces

Agrupa los enlaces de navegación internos o externos. Puede haber varios en una misma página. Estructura: "nav"

5.Sección de información

Organiza bloques de contenido relacionados temáticamente. Puede contener artículos, divs o subsecciones. Estructura: "section"

6.Artículo

Se usa para contenido independiente y específico como noticias o publicaciones. Ideal para textos que tienen sentido por sí solos. Estructura: "article"

7.Contenido adicional

Incluye información complementaria como anuncios, enlaces o notas. No está directamente relacionada con el contenido principal. Estructura: "aside"

8.Imagen

Permite insertar imágenes con pie explicativo opcional. Ideal para ilustraciones o gráficos destacados. Estructura: "figure"

9.Saltos de línea

Se utiliza para separar contenido en distintas líneas sin crear nuevos párrafos. Muy útil en textos continuos. Estructura: "br"

10.Párrafos

Agrupa bloques de texto que expresan ideas completas. Es la base de la estructura textual en HTML. Estructura: "p"

11.Sangrías

Define elementos dentro de listas de definición. No se usa para sangrías visuales, sino semánticas. Estructura: "dd"

12.Negritas

Resalta texto importante visualmente. No implica énfasis semántico, solo estilo. Estructura: "b"

13.Cursiva

Se usa para destacar palabras o frases, como títulos o términos extranjeros. Solo afecta al estilo visual. Estructura: "i"

14.Encabezados

Organizan el contenido jerárquicamente, desde el más importante (h1) al menos (h6). Son clave para el SEO y la accesibilidad. Estructura: "h1" a "h6"

15.Imágenes

Permite mostrar imágenes en la web desde una ruta específica. Debe incluir texto alternativo para accesibilidad. Estructura: "img"

16.Enlaces con imagen

Combina una imagen con un enlace, útil para logotipos o banners. Se usa junto con la etiqueta de enlace. Estructura: "img src='...' alt='...'"

17.Listas

Organizan contenido en forma de lista ordenada o desordenada.
  • Listas ordenadas: ol-/ol más li
  • Listas desordenadas: ul-/ul más li
Cada ítem se define con una etiqueta específica. Estructura: "ul"/"ol" + "li"

18.Formularios

Permiten la interacción del usuario con la página mediante campos de entrada. Estructura: "form" + "input" Incluyen campos como:
  • Tu nombre (input type="text" size="40" title="Escribe tu nombre real. Es el que aparecerá en tu..." name="nombre" autofocus required/),
  • Contraseña (input type="password" size="7" title="Escribe la contraseña que desee..." name="password" required/),
  • Correo electrónico (input type="email" placeholder="usuario@dominio.com" size="40" title="Escribe tu correo electrónico..." name="mail" required/),
  • Fecha de nacimiento (input type="date" value="2000-01-01" name="nacimiento" title="La fecha en la que naciste, ya que..."/), etc.

19.Tablas

  • Etiqueta principal en las tablas: table
  • Define una fila dentro de la tabla: tr
  • Define una celda de datos normal dentro de una fila: td
  • Define una celda de encabezado: th
  • Hay que destacar que para que salgan las líneas hay que usar css

20.Audio

Hay tres formatos complatibles con html que son: MP3, OGG y WAB.
La etiqueta que se usa es "audio".

21.Video

Permite insertar contenido audiovisual directamente en la página sin necesidad de plugins externos.
La etiqueta es "video", se puede añadir videos de dos tipos:
  • Con el archivo descargado,soporta aschivos de tipo: MP4, OGV, WebM. Puede incorporar atributos como:
  • "width y height": tamaño del vídeo.
  • "controls": añade barra de reproducción (play, pausa, volumen).
  • "autoplay": inicia automáticamente.
  • "loop": repite al finalizar.
  • "poster": imagen previa al inicio.
  • "muted": reproduce sin sonido.
  • Con un enlace de youtube mediante un "iframe".
